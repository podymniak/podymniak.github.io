2. How do I use it?

üõ†Ô∏è Getting Started

  1. Install addon from the Google Workspace and agree to the necessary access. Once installed, you will find the addon on the sidebar of your Google Calendar.
  2. When opening the addon for the first time, you will be prompted to provide an access token. This is a one-time setup process. The access token allows the addon to establish a connection with your ClickUp account.

After providing the access token and pressing enter, your tasks will be loaded and saved to your user properties (accessible only to you). With that, you can begin reporting your time entries üöÄ

NOTE: ClickUp Tasks are not loaded directly from the ClickUp platform on every click. You need to use the 'Update tasks list' button after assigning yourself to a new task. Only non-closed tasks that are assigned to you will be visible in the task selection section.


üìù Reporting

  1. Create a new event / click on an existing event in your Google Calendar.
  2. Find and click on your task in the task selection section.

‚ú® Your time is now reported! ‚ú®

Reported properties:

  • Start time - start time of the calendar event,
  • Duration - duration of the event,
  • Task - as selected from the addon dropdown,
  • Billable - based on your selection, OFF by defaul,
  • Entry description - calendar event title.

Events on which the addon is used will change colors. Event Status Colors:

  • Reported event - displayed in sage color.
  • Assigned event - displayed in lavender color.
  • Deleted entry - displayed in tomato color.


3. Additional features ü§∏‚Äç‚ôÄÔ∏è

  • If the meeting owner assigned it to a task, other participants will have that task visible as well, so no need for them to search for it*. You can also report time to a different task than the meeting‚Äôs owner without changing it for others.
  • For recurring meetings, the assigned task will be remembered after the first report. Assigned tasks can be reported by clicking the REPORT button.
  • The addon allows entry update for duration, description, billability, date, and task, as well as deletion of entries.
  • You can report your out-of-office events - addon will report 8h/day excluding weekends.

*Please note that your tasks or entries within the addon are not visible to other users. The only exception is when you, as the meeting organizer, report an event to a task that is not available to other meeting participants. In this case, they will be able to see the task's name but will not be able to report time there or open the task to see details.
